AadhaarBAS Installation Guide

Follow these simple steps to install AadhaarBAS without any issues.

Package Conflict Error Fix
1

If you see this error

"App not installed as package conflicts with an existing package"

If this error appears, simply uninstall the old AadhaarBAS app already installed on your device.
2

Download Again

After uninstalling the old AadhaarBAS application, download the latest version from this website and install it normally.

Enable Mock Location

Before using AadhaarBAS, make sure Mock Location is enabled.

  • Open Developer Options
  • Tap Select Mock Location App
  • Select your Mock Location App
  • Return and open AadhaarBAS
