AadhaarBAS Installation

Follow these easy steps to install the app perfectly on your device.

⚠️ Package Conflict Error Fix
1

Uninstall Old App

If you see the "App not installed as package conflicts with an existing package" error, you just need to uninstall the old AadhaarBAS app already installed on your device.

Note: You must completely uninstall the older version before installing the new one.
2

Download & Install

After uninstalling the old app, click the download button below to get the latest AadhaarBAS APK from this website and install it normally.

📍 Enable Mock Location

Make sure Mock Location is enabled in your phone settings before opening AadhaarBAS:

  • Go to Settings and open Developer Options
  • Scroll down and tap on Select Mock Location App
  • Select your Mock Location application
  • Now return to the home screen and open AadhaarBAS
