AadhaarBAS Setup Guide

Follow the installation steps below to avoid common installation issues.

Download APK
1

Uninstall Old AadhaarBAS

If any old AadhaarBAS version is installed on your device, uninstall it completely before proceeding.

Package Conflict Error occurs when an old version already exists on the device.
2

Install Official AadhaarBAS App

Download and install the latest official AadhaarBAS application from Play Store.

3

Enable Mock Location

Developer Options → Select Mock Location App → Choose the required Mock Location application.

4

Install AadhaarBAS Fix APK

After successful setup, install the provided Fix APK.

5

More Details → Install Anyway

Some devices may show a security warning during installation.

Click More Details → Install Anyway → Continue Installation
6

Open AadhaarBAS

Grant all permissions, enable location and verify that everything is working properly.

Frequently Asked Questions

Why am I getting Package Conflict Error?

An older version with the same package name already exists on your phone.

App is not installing?

Remove previous versions and try again.

Play Protect warning appears?

Click More Details and then Install Anyway.
